Model Comparison

This page explains the models used in the project. To generate classification analysis, upload a gait image on the Classifier page and click Compare All Models.

CNN

The CNN is the baseline model. It learns silhouette shape features directly from the training images using convolution, pooling, and dense classification layers.

ResNet / MobileNet

This is the transfer-learning comparison slot. The notebook saved MobileNet, so the app uses MobileNet as the fallback when a separate ResNet model is not available.

EfficientNetB0

EfficientNetB0 is a transfer-learning model. It compares the custom CNN against a pretrained architecture adapted for the three gait classes.
